# Todo Demo Project

Простий REST API для задач (ToDo List) на Node.js + Express + MongoDB.

**Функції:**
- Створення задач
- Перегляд всіх задач
- Оновлення задачі (completed)
- Видалення задачі

**Технології:**
- Node.js
- Express
- MongoDB
- Mongoose
- CORS

**Запуск:**
1. Встановити залежності: npm install
2. Створити .env файл з MONGO_URI
3. Запустити сервер: npm start
